## 🚀 Overview

**NEXA-Kernel** is a minimal, production-oriented execution kernel designed for building autonomous systems.

It provides:

- deterministic execution
- self-healing runtime
- intelligent routing (baseline)
- execution memory (v22)

> 🧠 Not an app — a foundation layer for building adaptive systems

---

## ⚙️ Core Capabilities

### 🔁 Self-Healing Runtime
- automatic retry on failure
- error detection and patch attempts
- execution stabilization

### ⚙️ Execution Engine
- isolated and deterministic execution
- pluggable structure
- safe executor layer

### 🧠 Intelligent Routing (v21)
- abstract task dispatching
- decouples execution from caller
- prepares for adaptive strategies

### 📦 Execution Memory (v22)
- stores past runs
- tracks results + scores
- enables learning from history
